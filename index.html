🍤 Seafood Amin

Sistem Pemesanan dan Kitchen Display untuk Restoran Premium

Aplikasi ini adalah backend PHP dan tidak bisa diakses langsung dari GitHub Pages.

Untuk menjalankan aplikasi:

  • Clone repository
  • Setup XAMPP atau server PHP lokal
  • Import database MySQL
  • Akses via http://localhost/seafood-amin
